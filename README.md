# Frontend Developer Assignment

## Description
This project is a solution for the Junior Frontend Developer assignment, showcasing the creation of a React application with Redux and TypeScript. The application features a user management table with advanced filtering capabilities.

## Technologies Used
- **React**: For building the user interface.
- **TypeScript**: For type safety and reducing runtime errors.
- **Redux**: For state management.
- **SCSS**: For styling the application.

## Features
- **User Management Table**: 
  - Displays user information (name, username, email, and phone) fetched from a mock API.
- **Advanced Filtering**: 
  - Search input fields for filtering the table dynamically by name, username, email, and phone.
- **State Management**: 
  - Managed using Redux.
- **Type Safety**: 
  - Ensured throughout the application using TypeScript.
